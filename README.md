### How do I get set up? ###

* no configuration needed
* need vuejs installed
* no database configuration
* no test sets
* add to config/app.php :

        Khofaai\Laraset\KhofoServiceProvider::class,
        Khofaai\Laraset\core\Providers\RouteServiceProvider::class,
        ...
